Setup Steps in Bubble.io

  1. Install API Connector plugin in your Bubble app
  2. Click Add another API → Name it "DataMind AI"
  3. Set Root URL: https://datamind-production.up.railway.app
  4. Add a new call: GET /bubble/predict/churn — mark parameters as dynamic
  5. Click Initialize call — Bubble auto-detects the response schema
  6. Use Get data from an external API in your workflows
  7. Map data.prediction.churn_probability to a Bubble text/number field
  8. Display data.ai_insight in a multi-line text element
